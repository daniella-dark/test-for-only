# test-for-only

Тестовое задание для компании Only.

---

## Стек технологий

- ReactJS
- TypeScript
- Webpack
- Swiper
- GSAP
- SCSS

## Установка и запуск

Установка зависимостей:\
`npm install`

Сборка проекта в режиме разработки и запуск локального сервера:\
`npm start`

Сборка бандла:\
`npm run build`
